My Story

It all started in my primary school, where they started making everyone do coding courses on scratch. Ever since I realised what I could make and do on scratch, I decided to continue and expand my interests. But soon, I got bored. I decided to do more advanced courses.

The journey didn't end there. I decided to take a Python Udemy course for kids where they made me make cool shapes using Python Idle. I decided it was way too basic for me and I pursued a Python course for adults. Alas, I took another Udemy course, a programming 100 days bootcamp. But studies got in the way and I desired to learn another programming language, one that could make websites.

I found Angela Yu's other course for the full web development course and I decided to try it. It is so fun making portfolios and websites which I can make via Visual Studio code and actually see on my browser. Although there are many AI tools to make websites my own, I have decided to code because I want to know how it works and to make me feel a satisfaction that what I make, is my own.
